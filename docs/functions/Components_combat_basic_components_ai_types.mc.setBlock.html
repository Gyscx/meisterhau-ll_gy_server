    • setBlock(
          pos: IntPos,
          block: string | Block | NbtCompound,
          tileData: number,
      ): boolean

      设置指定位置的方块

      通过此函数，将一个坐标对应的方块设置成另一个，类似于命令 /setblock

      Parameters

      • pos: IntPos

        目标方块位置

      • block: string | Block | NbtCompound

        要设置成的方块对象、方块标准类型名（如minecraft:stone）或方块NBT数据

      • tileData: number

        方块状态值，同原版 /setBlock 指令的 tiledata，默认为0，仅通过方块类型名放置方块时有效

      Returns boolean

      是否成功设置
