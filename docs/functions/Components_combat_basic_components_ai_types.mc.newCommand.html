    Function newCommand

    • newCommand(
          cmd: string,
          description: string,
          permission?: PermType,
          flag?: number,
          alias?: string,
      ): Command

      注册一条顶层命令

      这里提供了注册自定义命令的接口。

      通过对接 BDS 内置的命令系统，你注册的命令可以由玩家、控制台、命令方块、NPC等各种游戏中可以执行命令的对象所使用，
      在 addon 中，也可以使用这里所注册的命令。

      Parameters

      • cmd: string

        待注册的命令

      • description: string

        命令描述文本

      • Optionalpermission: PermType

        指令执行所需权限，默认为PermType.GameMasters

      • Optionalflag: number

        默认值 0x80，目前直接按此输入即可，后续会进行相关修改

      • Optionalalias: string

        命令的别名

        可以为命令设置多个别名，执行的时候相当于触发同一条命令
        （这里只能填一个，建议使用Command.setAlias()）

      Returns Command

      指令对象
